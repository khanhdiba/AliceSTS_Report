# AliceSTS_Report
This is the specialized project report for the group with ID HK251-DAGD1-183 from HCMUT - VNUHCM. This included the system design and analysis for the system, which is implemented in the second phase for the capstone project.

# What is Alice?
Alice is the second phase of the Smart Scheduler AI project, we aim to provide a mobile application that can help user to manage his/her schedule, task, and health. We also integrate Machine Learning to improve the user experiences.

# Progress
In this repository, we only have the system design and analysis. The implementation will be done in the following semester with the name Alice application.
